You can think of Florida’s SARnet statewide repeater network as a “mega horn” for ham radio nerds. Imagine you’re in the caboose of a loose-knit caravan stretching 200 miles across the Panhandle heading to Key West. Amateur radio operators in your caravan could stay in touch using any SARnet repeater that broadcasts to all SARnet repeaters across the state.

There’s just one problem: SARnet’s website & charts lack ease-of-use. Sure, you can download a Chirp file from RepeaterBook and upload it to your mobile unit, but then what?

This project shows how to use Chirp, scissors, tape, and 3x5 index cards to improve mobile amateur radio on Florida’s interstates & turnpike.

The new "Yaesu_RT.header" file gives you the .csv header you need for Yaesu's and RT Systems' software. USE A TEXT EDITOR, not a word processor, to replace the first line in a Chirp .csv file with the line in Yaesu_RT.header. Yaesu's and RT Systems' software will recognize all the fields it needs when you "import" the file.
